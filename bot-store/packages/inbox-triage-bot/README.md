# Inbox Triage Bot

**O que faz**
- Transformar um INBOX caótico em: Next / Scheduled / Waiting.

## Regras (simples, úteis)
- Se tem data/hora explícita → Scheduled
- Se depende de outra pessoa/sistema → Waiting
- Se dá para fazer em <15 min → Next
- Se não é ação → Ideia/Nota

## Runbook
1) Importa INBOX.md
2) Marca 10 items mais importantes
3) Reescreve cada item como verbo+resultado
4) Move para o bucket certo
